In this project, the DIA mass spectrometry data were analyzed using the search software DIA‑NN (v1.8.1) with the library‑free method. The search parameters were as follows: the database used was uniprotkb_proteome_UP000005640_human_82493_20240528.fasta (containing 82,493 sequences); the option for deep‑learning‑based prediction of a spectral library was enabled; Match Between Runs (MBR) was activated to generate a spectral library from the DIA data, which was then used to re‑analyze the DIA data for protein identification and quantification; both precursor‑level and protein‑level identifications were filtered at a 1% False Discovery Rate (FDR). The filtered data were subsequently used for follow‑up bioinformatics analysis.
